通常のJavaScript

function addTodo(text) { 
  return { 
    type: "ADD_TODO", 
    payload: { 
      id: Date.now(), 
      text: text, 
      completed: false 
    } 
  }; 
} // 使用例
dispatch(addTodo("買い物に行く"));

上記のコードは、Redux/NgRxで使用されるアクションクリエイター関数の例です。この関数は、TODOアイテムを追加するためのアクションオブジェクトを生成します。

アクションオブジェクトには、typeプロパティとpayloadプロパティが含まれています。payloadには、TODOアイテムのID、テキスト、完了状態が含まれています。

このアクションクリエイターを使用するには、dispatch関数に渡します。例えば、dispatch(addTodo("買い物に行く"))のように使用します。
